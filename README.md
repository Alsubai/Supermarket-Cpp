Supermarket System

Project Name:

Supermarket System

Description:

A console-based program that simulates a simple supermarket checkout. Users can:
	•	Select products from a menu
	•	Enter quantity for each product
	•	Receive discounts on selected items
	•	Buy multiple products in one session
	•	Get a final bill with the total amount

The program includes special offers, such as a second sandwich at half price and general discounts on certain products.

Features:
	•	Menu-driven interface for easy navigation
	•	Handles product selection and quantity input
	•	Applies discounts based on product and quantity
	•	Supports multiple product purchases per session
	•	Displays the final bill in a clear format

How to Run:
	1.	Compile the code using any C++ compiler.
	2.	Run the executable in the terminal or console.
	3.	Follow the on-screen menu to select products and quantities.
	4.	After finishing, view the total bill at the end.
